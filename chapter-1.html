海滩的边界

首个开放区域 · 南部海岸 · 2个限时成就

📋 详细流程
1. 离开序章区域，进入南部海岸开放地图
2. 完成3个教学配送（药品/食物/工具到附近据点）
3. 首次遭遇MULE营地，学习潜行绕后或正面战斗
4. 【重要】在离开区域前，前往东南角悬崖边寻找NPC'老艾萨克'，对话两次（触发成就：搁浅者的回响）
5. 【重要】在BT区域西北侧的洞穴内，寻找紫色发光脐带并破坏（触发成就：海滩的第一条脐带）
6. 收集区域内散落的3个记忆芯片
7. 最后交付主线任务道具到港口，进入新区域（此后无法返回）
1. Leave prologue area, enter south coast open map
2. Complete 3 tutorial deliveries (medicine/food/tools to nearby outposts)
3. First MULE camp encounter, learn stealth flanking or direct combat
4. [IMPORTANT] Before leaving, go to southeast cliff and talk to NPC 'Old Isaac' twice (Trophy: Echoes of the Stranded)
5. [IMPORTANT] In NW cave of BT zone, find and destroy purple glowing umbilical cord (Trophy: First Umbilical of the Beach)
6. Collect 3 scattered memory chips in the area
7. Finally deliver main quest item to port, enter new area (cannot return after)
可获得物品：
  • 血液手雷正式配方
  • 攀爬柱（可重复使用）
  • 梯子（可重复使用）
  • 记忆芯片#01
  • 记忆芯片#02
  • 记忆芯片#03
  • 开罗尔晶体x5
Items:
  • Hemogrenade Formal BP
  • Climbing Anchor (reusable)
  • Ladder (reusable)
  • Memory Chip #01
  • Memory Chip #02
  • Memory Chip #03
  • Chiral Crystal x5
⚔️ BOSS战攻略 · 异形BT
📊 阶段分解：
  • 阶段1 - 地面爬行：保持距离，用血液手雷攻击，注意躲避冲撞
  • 阶段2 - 站立形态：攻击前摇明显，侧向闪避后反击，弱点在胸口发光处
🎒 推荐装备： 血液手雷x5、攀爬柱、修复喷雾x2
💡 特殊机制： BOSS被血液手雷击中后会短暂硬直，可以绕后攻击。场景中有补给箱可以补充血液手雷。
⚔️ Boss Guide · Aberrant BT
📊 Phases:
  • Phase 1 - Ground crawling: Keep distance, use hemogrenades, dodge charges
  • Phase 2 - Standing form: Clear attack windup, side dodge then counter, weak point is glowing chest
🎒 Recommended Gear: Hemogrenade x5, Climbing Anchor, Repair Spray x2
💡 Special Mechanics: BOSS briefly stuns when hit by hemogrenade, can attack from behind. Supply boxes in arena refill hemogrenades.
🔍 隐藏物品/彩蛋：
  • 隐藏道具「老艾萨克的怀表」：与NPC对话两次后，在他旁边的箱子里获得
  • 隐藏饰品「海滩贝壳」：西北侧洞穴最深处的石缝中
🔍 Hidden Items/Easter Eggs:
  • Hidden item 'Old Isaac's Pocket Watch': After talking twice, find in box next to him
  • Hidden accessory 'Beach Shell': In deepest part of NW cave, between rocks
🏆 本章成就/奖杯：
  • 🏆 搁浅者的回响（银杯） - 与老艾萨克对话两次（不可错过）
  • 🏆 海滩的第一条脐带（银杯） - 破坏紫色脐带（不可错过）
  • 🏆 初出茅庐（铜杯） - 完成第一章所有配送任务
🏆 Trophies/Achievements:
  • 🏆 Echoes of the Stranded (Silver) - Talk to Old Isaac twice (Missable)
  • 🏆 First Umbilical of the Beach (Silver) - Destroy purple umbilical cord (Missable)
  • 🏆 Novice Porter (Bronze) - Complete all deliveries in Chapter 1
🎮 平台差异提示：
PS5: 推荐性能模式(60fps)，BT区域帧数更稳定；PC: 开启DLSS/FSR质量模式可大幅提升BT区域流畅度。
🎮 Platform Differences:
PS5: Performance mode (60fps) recommended for stable BT zone frames; PC: Enable DLSS/FSR Quality mode for better BT zone performance.

⚠️ 两个隐藏成就必须完成，离开第一章后无法返回！成就1：与悬崖边NPC对话两次；成就2：破坏洞穴内的紫色脐带。离开前请务必确认已完成。
